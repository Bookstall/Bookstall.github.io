## SRT 字幕文件

字幕最常用的格式是 `SRT`，它是纯文本内容以及涵盖了字幕的开始和结束时间信息。因此，如果您有一些想要录制的视频以及字幕要与在线字幕一起上传，则可以将简单的文本文件转换为 `SRT` 格式。

SRT 是 SubRip 字幕文件，以 SubRip 软件命名，该软件是开源的，可以免费使用。SRT 格式与几种视频播放程序和软件兼容。SRT 文件包含与字幕有关的信息以及字幕内容的开始和结束时间。
